Frequently Asked Questions

Is my data safe?

Yes! All JSON processing happens entirely in your browser. We never see, store, or transmit your data.

Is json2schema.app really free?

Yes, completely free with no registration required. We support the tool through ethical advertising.

Can I use generated code in commercial projects?

Absolutely! All generated code is yours to use however you like, including commercial projects.

Will you add support for [language]?

We're always looking to add more languages! Send us a feature request with your suggestion.

I found a bug. How do I report it?

Use the contact form above and select "Bug Report" as the subject. Include steps to reproduce the issue.
